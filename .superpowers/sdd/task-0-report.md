## 당시 미확인 항목과 현재 처리 상태

- 당시에는 연결된 마이그레이션 이력을 읽지 않았으나, 이후 Supabase 플러그인으로
  `tips dashboard` 프로젝트와 적용 이력을 읽기 전용으로 확인했습니다.
- 로컬 Docker·PostgreSQL·Supabase 실행 환경이 없어 실제 DB pgTAP은 실행하지 못했습니다.
  2026-07-16 실행 방침에 따라 이는 개발 중단 사유가 아니며, 저장소의 SQL 계약 테스트와
  플러그인 읽기 검증으로 개발을 계속합니다.
- 당시 잠시 확인한 개발 서버는 의존성 복구 과정에서 중단됐습니다. 현재 구현 작업 트리의
  개발 서버는 `http://localhost:3001`에서 별도로 실행·확인하고 있습니다.
- 당시에는 브라우저 제공자 차단 장부를 확립하지 않았습니다. 외부 제공자 호출은 시도하지 않았고,
